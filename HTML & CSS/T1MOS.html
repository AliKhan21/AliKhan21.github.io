Microsoft Operating Systems


This course focuses on Microsoft operating systems. Students learn the components of an operating system and learn to interact with a PC using a variety of Microsoft products. Additionally, students learn to customize and manage an operating system as well as solve software-level difficulties.

  • I learned the key differences between Windows, Mac OS and Linux.
  • I learned how to use command prompt/DOS.
  • I learned how to use PowerShell.
  • I learned how PowerShell is much more advanced/powerful then DOS.
  • I learned how to create programs through DOS and PowerShell.
