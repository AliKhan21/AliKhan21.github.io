Web Development


In this course students learn to use HTML, CSS, and JavaScript to develop well-designed Web pages. Students learn to apply appropriate techniques and to include forms, images, and tables.

  • I learned how to create web pages with HTML.
  • I learned the basics of CSS and how to add some styles to my web pages.
  • I learned how to create tables with HTML.
  • I learned how to create lists with HTML, ordered and Unordered.
  • I learned how to insert images in HTML.
