Microsoft Integration


In this course students learn to work efficiently with multiple applications using the automation and integration capabilities of Microsoft Office. PowerPoint, advanced features of Word, and the Microsoft mini-applications are also covered.

  • I learned the difference between verbs, nouns, pronouns, adverbs, and adjectives.
  • I learned the differences between commands, statements, and questions.
  • I learned how to properly deal with stress in a work environment.
  • I learned the differences between simple, compound, and complex sentences.
  • I learned how to identify and fix run-on sentences and comma splices.
